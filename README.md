# Fluvit

_A hydraulic erosion simulation_

Gahwon Lee

### Compiling

```
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
make -j8
```

### How to Run
* `./fluvit -h` for help
* `./fluvit path/to/png` to load heightmap
* `./fluvit` for random terrain


### Controls

Right click and WASD to fly
